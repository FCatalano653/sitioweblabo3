  1. Dar un par de ejemplos de uso de etiquetas meta

    Un ejemplo de uso de etiquetas meta puede ser el siguiente:

    <meta charset="UTF-8">

    El cual se utiliza para especificarle al navegador la codificacion de caracteres que debe utilizar.


    Otro ejemplo:

    <meta http-equiv="X-UA-Compatible" content="IE=edge">

    En este caso se le indica al navegador el modo de compatibilidad que por ejemplo es requerido con Edge de Microsoft.

  2. Nombrar al menos dos atributos aplicables a la etiqueta HTML

    Atributos aplicables a la etiqueta html :

    • lang=""

      Especifica el idioma del texto contenido de un documento html.


    • id=""

      Este atributo define un identificador unico propio de cada elemento.


    • class=""

      Se utiliza para aplicar estilo a determinador elementos que comparten propiedades con el mismo valor.

  3. ¿Como se relaciona un header http con la etiqueta head en HTML?

    No tienen ningun tipo de relacion mas que denominarse de una forma similar ya que el header http es el que contiene el valor del juego de variables propio de cada cliente, en cambio la etiqueta head es un contenedor de otras etiquetas como las meta, entre otras, que aportan informacion util para el navegador.

  4. ¿Qué es UTF8?

    El UTF-8 es un formato de codificacion de texto o caracteres perteneciente a Unicode, el cual incluye mas de 120.000 simbolos de diferentes alfabetos e idiomas, asi como tambien emojis. Es un estandar de la web que permite una inclusion mayor a sus predecesores.

  5. ¿Qué atributo HTML sirve para aplicar estilo? ¿A qué etiquetas es aplicable?

    El atributo style="" sirve para aplicar estilo a una determina etiqueta y se puede denominar como atributo global, ya que es aplicable a todas las etiquetas HTML.

  6. ¿Donde se indica que un enlace debe ubicar el documento en una nueva pestaña?

    A traves del atributo target="_blank" se le indica a la etiqueta de ancord que el marco de destino de un vinculo es en una pestaña. El valor predeterminado es "_self".

  7. Explicar cuando usarian etiquetas semanticas y con que proposito

    Las etiquetas semanticas se usan para darle un significado y jerarquia al contenido de nuestro documento html o pagina web, es decir que contribuyen a la coherencia de nuestro contenido de texto. Ademas su proposito tambien esta vinculado a que su correcta utilizacion es tenida en cuenta para los motores de busqueda posicionen mejor nuestro sitio.

  8. ¿Qué atributos de estilo basicos aplicarian a una tabla? Explicar

    Los atributos de estilo basicos que se aplican a una tabla puede ser "border:solid". El cual determina el estilo visual del borde de una tabla, por ejemplo en este caso el borde seria solido.

    Otros atributos pueden ser el "width:00px" o el "height:00px", los cuales determinan el ancho y alto de la tabla respectivamente.

  9. ¿Qué diferencia existe entre la etiqueta body de un documento HTML y el body del protocolo HTTP?

    La etiqueta body de un documento HTML es dentro de la cual se declara todo el contenido del documento, en cambio el body del protocolo HTTP contiene el texto que representa el documento html en si, archivos "adjuntos" o fragmentos de video en caso de streaming.

  10. ¿Qué atributos marcan la agrupacion de celdas de una tabla? ¿A qué etiquetas se aplica?

    Los atributos que marcan la agrupacion de celdas de una tabla son colspan y rowspan. El primero se utiliza para determinar cuantas posiciones avanzara sobre las siguientes columnas, es decir de forma horizontal y la segunda cuantas posiciones avanzara de forma vertical. Ambas se aplican a la etiqueda <td>.
